# Orval'IA Studio - Site Estático

Este projeto é um site estático (HTML/CSS) otimizado para o GitHub Pages.

## Configuração do GitHub Pages (Sem ramificações extras)

Para rodar este site diretamente da branch principal (`main`):

1. Acesse seu repositório no GitHub.
2. Vá em **Settings** -> **Pages**.
3. Em **Build and deployment** -> **Branch**:
   - Selecione `main` (ou a branch principal que você usa).
   - Verifique se a pasta está como `/(root)`.
4. Em **Custom domain**, insira `orvalia.com.br`.
5. Clique em **Save** e aguarde a propagação do DNS.

## Edição
Qualquer alteração feita no arquivo `index.html` será refletida automaticamente no seu domínio após o salvamento no GitHub.
